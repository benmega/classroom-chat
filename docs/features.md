# Features

## Real-Time Chat
- Built using Flask-SocketIO for efficient communication.
- Detects URLs for educational purposes, such as CodeCombat links.

## Challenge Tracking
- Challenges are assigned unique slug names for easy reference.
- Teachers can define challenge point values.

## Duck System
- Ducks are awarded upon challenge completion.
- Real-time updates to profile and leaderboard.

## Leaderboards
- Displays top 10 performers with duck counts.
- Sorted dynamically based on achievements.

## Customizable Profiles
- Users can edit their username and password.
- Ducks and achievements displayed prominently.
