# ISSUE-009: Non-Responsive "New Conversation" Button

## Status: Open
## Priority: Medium
## Category: Functional / Chat

### Description
In the Chat sidebar, clicking the "+" (plus) button next to the "Messages" title does nothing. No modal appears, and no new conversation is initialized.

### Steps to Reproduce
1. Log in.
2. Navigate to Chat.
3. Click the "+" button in the sidebar.

### Expected Behavior
A modal or dropdown should appear to select users or name a new channel/conversation.

### Actual Behavior
The button has a hover effect but no click handler or functional result.
