# ISSUE-013: Minimalist Design (Non-Premium)

## Status: Open
## Priority: High
## Category: Design / Aesthetics

### Description
The overall application design is highly minimalist and feels like a functional MVP rather than a "premium" product. It lacks vibrant colors, cohesive branding, subtle gradients, micro-animations, and advanced typography that would "WOW" the user.

### Key Areas for Improvement:
- **Login Page**: Very sparse, lacks background elements or a brand logo.
- **Bit Shift Page**: Extremely basic layout with plain buttons.
- **Profile Stats**: Just text on a white background.
- **General**: Lack of consistent shadows (`var(--shadow-lg)` is defined but underutilized) and transitions.

### Expected Behavior
A premium UI with rich aesthetics, modern typography (Outfit/Inter), and interactive flair (hover effects, smooth transitions).

### Actual Behavior
A clean but very "standard" Bootstrap-like or basic Tailwind-like appearance.
