# ISSUE-006: Chat Message Placeholder Alignment

## Status: Open
## Priority: Low
## Category: UI / Chat

### Description
In the Chat interface, the circular placeholder containing user initials (e.g., "Mr.") is poorly aligned with the message bubble. It appears to overlap the bubble border or sit at an awkward offset, detracting from the "premium" feel.

### Steps to Reproduce
1. Log in.
2. Navigate to Chat.
3. Look at any message sent by a user without a profile picture.

### Expected Behavior
The avatar/initials circle should be vertically aligned with the bottom or top of the message bubble with consistent spacing.

### Actual Behavior
The circles are misaligned and overlap the message content/bubble.
