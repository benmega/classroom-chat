# Mobile Bit-Shift Layout (Responsive Layout)

## Description
The denomination input grid (1b, 10b, 100b, etc.) on the Bit-Shift page becomes extremely cramped and loses proper alignment on mobile widths (500px or less).

## Steps to Reproduce
1. Log in as any user.
2. Navigate to the Bit-Shift page (`/bit-shift`).
3. Resize the browser to mobile width (500px).
4. Observe the grid of denomination cards.

## Expected Result
The cards should stack or resize gracefully to fit comfortably on smaller screens.

## Actual Result
The grid becomes overly dense and misaligned, making interactions difficult.

## Impact
Medium - Affects the usability of the Bit-Shift feature on mobile.
