# Admin Panel Owl Mascot Image Oversized

## Description
The large owl mascot image displayed in the Admin Panel sidebar is oversized and breaks the intended layout proportions.

## Steps to Reproduce
1. Navigate to the Admin Panel.
2. Look at the sidebar or the area containing the owl mascot image.
3. Observe the image dimensions relative to its container.

## Expected Result
The mascot image should be appropriately scaled to fit within the sidebar without dominating the layout.

## Actual Result
The image is oversized, throwing off the visual balance of the Admin Panel.

## Impact
Low - A visual and aesthetic bug that does not break functionality but reduces the professional feel.
