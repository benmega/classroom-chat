# Achievements Page Filter Alignment and Redundant Header

## Description
The `/achievements` page has several UI/UX issues that detract from a premium feel:
1. The "Hall of Achievements" header card is large and centered, creating redundant vertical space.
2. The search bar and filter chips ("All", "Ducks", "Certificate", etc.) are left-aligned below the centered header, creating a visual disconnect.
3. The page would benefit from a more compact, aligned layout.

## Steps to Reproduce
1. Navigate to `/achievements`.
2. Observe the alignment of the header stats versus the filter chips below.

## Expected Result
The header elements and filters should be visually aligned (e.g., all centered or all left-aligned) and the layout should feel cohesive and modern.

## Actual Result
Centered header stats with left-aligned filters below create a visually unbalanced layout. The large header card is also considered redundant.

## Impact
Medium - Affects the aesthetic quality and professional feel of the achievement system.
