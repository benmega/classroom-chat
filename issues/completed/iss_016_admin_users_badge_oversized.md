# Oversized Account Type Badges in User Management

## Description
The account type badges (e.g., "Student", "Administrator") in the User Management table are disproportionately large. The background box for these badges occupies too much vertical and horizontal space compared to the text content and other table columns, leading to a cluttered and unbalanced look on desktop resolutions.

## Steps to Reproduce
1. Log in as an Administrator.
2. Navigate to the Admin Panel.
3. Click on the "Users" link in the sidebar.
4. Observe the "Account Type" column in the users table.

## Expected Result
Badges should have a compact, refined design with appropriate padding that complements the table row height and text size, maintaining a premium aesthetic.

## Actual Result
Badges are excessively large, with large background boxes that make the table look unpolished and unbalanced.

## Impact
Medium - Negatively impacts the "premium" feel of the administrative interface and reduces data density in the table.
