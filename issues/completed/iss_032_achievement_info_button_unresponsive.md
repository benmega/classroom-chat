# Bug: Achievement 'Info' Button Unresponsive

## Description
Clicking the "Info" button on achievement cards in the `/achievements` gallery does not trigger any action. No modal appears, no navigation occurs, and there is no feedback to the user.

## Steps to Reproduce
1. Log in as a student.
2. Navigate to `/achievements`.
3. Click the "Info" button on any achievement card.

## Expected Result
A modal or tooltip should appear detailing the achievement requirements, or the user should be taken to a details page.

## Actual Result
Nothing happens.

## Impact
**Minor.** Reduces the utility of the achievements system.
