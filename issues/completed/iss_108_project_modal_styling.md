# iss_108_project_modal_styling
**Status**: Open
**Priority**: Minor
**Type**: UI/UX

## Description
The project view modal layout needs refinement to better utilize available screen space on desktop resolutions.

## Requirements
- Make the project modal slightly wider overall.
- Modify the technical layout logic for the code block: the code block height should naturally scale to match the combined height of the description and instructor review on the left-hand side.
- Prevent the code block from arbitrarily stretching and lengthening the modal beyond the height of its adjacent content.
