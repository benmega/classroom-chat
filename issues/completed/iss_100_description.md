# Parent Dashboard Header Container is Misaligned on Mobile

## Description
The top header section ("Your Children's Progress" and action buttons) on the Parent Dashboard is not centered and appears shifted to the left, losing its left margin/padding on mobile viewports.

## Steps to Reproduce
1. Log in as a Parent.
2. Navigate to the Parent Dashboard on a mobile viewport (e.g., width 390px).
3. Observe the top white container with the heading "Your Children's Progress".

## Expected Result
The header container should be centered with even margins on both the left and right sides.

## Actual Result
The container is shifted to the left, causing its left edge to touch the screen border while leaving a large gap on the right, breaking the visual layout.

## Impact
Medium - The layout looks unpolished and breaks aesthetic guidelines for mobile devices.
