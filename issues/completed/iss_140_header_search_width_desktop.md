# Header Search Bar Excessive Width on Desktop

## Description
The global search bar in the application header is excessively wide on large desktop resolutions (e.g., 1440px and above). This creates a visual imbalance and makes the header feel "empty" despite the large input field.

## Steps to Reproduce
1. Log in and navigate to the Dashboard.
2. View the application on a desktop resolution (1440px width).
3. Observe the "Search users..." input in the center of the header.

## Expected Result
The search bar should have a reasonable `max-width` to maintain visual balance and focus, centered in the header.

## Actual Result
The search bar stretches significantly, taking up a large portion of the header's center area.

## Impact
Low - This is an aesthetic enhancement to improve the "premium" feel of the desktop experience.
