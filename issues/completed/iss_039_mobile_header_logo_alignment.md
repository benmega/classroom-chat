# Mobile Header Alignment (Responsive Layout)

## Description
The header elements (logo and profile toggle) are pushed to the extreme edges of the layout on narrow screens (500px and 375px), leaving a disproportionately large gap in the center.

## Steps to Reproduce
1. Log in to the application as any user.
2. Resize the browser window to mobile width (500px or less).
3. Observe the logo on the far left and the profile toggle on the far right.

## Expected Result
The header elements should feel balanced and properly centered or spaced within a narrower container padding.

## Actual Result
A large gap exists, making the top bar feel disconnected and non-premium.

## Impact
Low/Medium - Affects the mobile responsiveness and premium feel of the app.
