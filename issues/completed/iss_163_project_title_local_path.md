# Project Title/Description Renders Local Path

## Description
In the portfolio/projects section, the project title or description incorrectly displays the full local file path (e.g., `C:\Users\Ben\...`) instead of the actual user-defined title.

## Steps to Reproduce
1. Navigate to the Projects/Portfolio page.
2. Observe the listed projects.
3. Check the text rendered for the project title or description.

## Expected Result
The actual title and description of the project should be displayed.

## Actual Result
A full local file path is displayed instead of the title/description.

## Impact
Medium - Exposes local environment details and breaks the expected visual presentation.
