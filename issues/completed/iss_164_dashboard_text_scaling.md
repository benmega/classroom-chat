# Dashboard Text Scaling Issues

## Description
The "Welcome back" text on the dashboard does not scale down appropriately for mobile viewports. It remains very large and takes up significant vertical space.

## Steps to Reproduce
1. Authenticate and open the application.
2. Resize the viewport to a mobile width (e.g., 390px).
3. Navigate to the Dashboard.
4. Observe the "Welcome back, [Name]!" heading text.

## Expected Result
The font size should scale down using responsive typography so that it fits neatly on a smaller screen.

## Actual Result
The font size remains very large (H1 style), dominating the top half of the screen.

## Impact
Low - UI Consistency. It's legible but feels unpolished.
