# Header Profile Dropdown Overlaps and is Cut Off

## Description
The user profile dropdown menu in the header overlaps with the right-side content or screen edge, causing the "Logout" button (and potentially other items) to be partially cut off.

## Steps to Reproduce
1. Navigate to any page with the main header.
2. Click on the user profile icon in the top right to open the dropdown menu.
3. Observe the positioning of the dropdown.

## Expected Result
The dropdown menu should be fully visible, positioned correctly without overlapping or being cut off by the screen edge.

## Actual Result
The dropdown is misaligned, causing the right side of the menu, including the "Logout" button, to be clipped.

## Impact
Medium - Degrades the premium aesthetic and makes logging out or accessing profile options difficult.
