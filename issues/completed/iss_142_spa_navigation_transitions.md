# Missing SPA Navigation Transitions

## Description
Transitions between main application views (e.g., Dashboard to Chat, or Profile to Achievements) are instantaneous and abrupt. Adding subtle fade or slide transitions would improve the flow and polished feel of the Single Page Application.

## Steps to Reproduce
1. Log in to the application.
2. Rapidly navigate between sidebar items or profile menu links.
3. Observe the lack of visual continuity or transitions between pages.

## Expected Result
Smooth, subtle transitions (e.g., 200ms fade-in) when switching views.

## Actual Result
Pages snap into view immediately.

## Impact
Low - Aesthetic/UX improvement to align with modern web design standards.
