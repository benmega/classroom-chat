# Chat UI Inconsistent Spacing

## Description
The Chat interface has significant layout and spacing issues. The message list is pushed far to the left, the input field is centered, and there are excessively large gaps between elements.

## Steps to Reproduce
1. Navigate to the Chat module.
2. Observe the layout of the message history and the text input area.

## Expected Result
Messages should be well-aligned (e.g., left for received, right for sent), and the input area should span the appropriate width with balanced padding and margins.

## Actual Result
The message list is far left, the input is centered awkwardly, and there are large, unappealing gaps in the layout.

## Impact
Medium - Severely impacts the usability and visual quality of a core feature.
