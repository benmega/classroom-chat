# Achievements Filter Alignment Issue

## Description
The filter chips on the Achievements page are not properly aligned with the search bar above them. There is a noticeable offset where the chips container has more left padding/margin than the search input field.

## Steps to Reproduce
1. Log in as any user.
2. Navigate to the Achievements page (/achievements).
3. Observe the alignment of the "Search badges..." input field and the row of filter chips (All, Ducks, Certificate, etc.) below it.

## Expected Result
The filter chips should be left-aligned with the search input field to create a clean, structured layout.

## Actual Result
The filter chips are slightly offset to the right compared to the search bar's left edge.

## Impact
Medium - Affects the visual consistency and perceived quality of the Hall of Achievements UI.
