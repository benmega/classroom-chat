# Oversized Ducks and Packets Indicators in Header

## Description
The "Ducks" and "Packets" status indicators in the main application header are disproportionately large on desktop viewports (1440px). They occupy a significant portion of the header height and appear too prominent compared to other navigation elements and the brand logo, disrupting the visual hierarchy.

## Steps to Reproduce
1. Log in to the application.
2. Observe the main header at the top of the page.
3. Note the size of the status cards for "Ducks" and "Packets".

## Expected Result
Status indicators should be compact and integrated smoothly into the header layout, providing information without dominating the visual space.

## Actual Result
Indicators are very large and visually heavy, appearing out of scale with the rest of the header components.

## Impact
Medium - Impacts the overall aesthetic balance and "premium" feel of the main user interface.
