# Dashboard Aesthetic Improvements and Polish

## Description
The main dashboard and chat sidebar have several "low-polish" elements that detract from the premium feel:
1. **Placeholder Truncation**: The search bar in the chat sidebar has a truncated placeholder text "Search conversatio".
2. **Generic Avatars**: User avatars are rendered as plain beige squares, which feels unfinished.
3. **Peripheral Glow**: The blue outer glow around the main application container feels dated.
4. **Button Icon Scaling**: Icons in the header (e.g., the duck in the "DUCKS" button) are disproportionately small.

## Steps to Reproduce
1. Navigate to the main dashboard/chat view.
2. Observe the sidebar search bar and the general aesthetic of the container and header.

## Expected Result
A cohesive, modern UI with appropriate text scaling, high-quality placeholders, and professional-grade effects (e.g., subtle shadows instead of strong outer glows).

## Actual Result
Several minor visual inconsistencies and dated design choices.

## Impact
Low/Medium - Affects the "wow factor" and premium perception of the app.
