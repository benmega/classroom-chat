# Bug: Admin Sub-page Header/Back Button Overlap

## Description
On various admin sub-pages (e.g., Pending Users, Projects), the "Back to Dashboard" button overlaps with the main page header text when the viewport is narrowed (and even at some standard desktop widths), creating a cluttered and unprofessional UI.

## Steps to Reproduce
1. Log in as Admin.
2. Go to Admin HQ -> User Approvals (or any sub-page).
3. Observe the top left area of the content panel.

## Expected Result
The "Back to Dashboard" button and the page title should have sufficient spacing and not overlap.

## Actual Result
Elements overlap.

## Impact
**Minor (Visual).** Affects the professionalism and readability of the Admin Panel.
